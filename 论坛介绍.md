## 🐷 肘子牌文章提取器 — 一键保存网页为 Markdown

逛论坛/看博客时看到好文章，想保存下来离线读或者收藏？我写了个 Chrome 扩展，一键提取正文，直接保存为 Markdown 文件。

### 功能

- **一键保存** — 点工具栏图标 → 保存当前页面，不需要复制粘贴
- **干净正文** — 用 Mozilla Readability 算法提取，去广告去导航只留正文
- **Markdown 输出** — 自动转成 MD 格式，带 YAML 头（标题、来源、日期）
- **可设保存路径** — 默认存下载目录，也能指定子文件夹
- **保存日志** — 成功失败都有记录，可导出为 MD 文件

### 安装

1. 下载 ZIP 或 clone 代码
2. 打开 Chrome → `chrome://extensions/`
3. 右上角开「开发者模式」
4. 点「加载已解压的扩展」→ 选择文件夹
