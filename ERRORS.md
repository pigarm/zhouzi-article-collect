# 肘子 错误参考文档

> 最后更新：2026-06-02

本文档列出本扩展保存文章时可能遇到的所有错误码、原因分析和建议。

---

## 错误码速查

| 错误码 | 含义 | 严重程度 |
|--------|------|---------|
| `NO_CONTENT` | 页面无可提取正文 | ⚠️ 常见 |
| `NOT_AN_ARTICLE` | 页面不是文章类型 | ⚠️ 常见 |
| `PAGE_TOO_SHORT` | 正文内容过短 | ⚠️ 常见 |
| `BLOCKED_BY_LOGIN` | 需要登录/付费 | ⚠️ 常见 |
| `EMPTY_PAGE` | 页面内容为空 | ⚠️ 偶发 |
| `CONTENT_SCRIPT_NOT_READY` | 内容脚本未加载 | ⚠️ 常见 |
| `DOWNLOAD_FAILED` | 文件下载失败 | ⚠️ 偶发 |
| `BLOCKED_BY_ROBOTS` | 被网站反爬机制屏蔽 | 🔴 少见 |
| `UNSUPPORTED_URL` | 不支持的链接类型 | 🔴 少见 |
| `TIMEOUT` | 操作超时 | 🔴 少见 |
| `UNKNOWN` | 未知错误 | 🔴 极少 |

---

## 详细说明

### NO_CONTENT — 页面无可提取正文

**触发条件**：
- Readability 算法无法在页面中找到文章正文
- 页面结构不符合文章模式（无 `<article>`、无密集 `<p>` 段落等）

**典型场景**：
- 视频页面（B站、YouTube）
- 图片画廊
- 网站首页/导航页
- 搜索结果页
- 评论区/论坛帖子列表

**建议**：
- 确认是否确实为一篇文章
- 如果页面是动态加载内容（SPA），等待内容完全渲染后再试
- 尝试用浏览器的「阅读模式」查看是否有内容

---

### NOT_AN_ARTICLE — 页面不是文章类型

**触发条件**：
- 页面分类/标签/结构不符合文章特征

**典型场景**：
- 电商商品详情页
- 社交媒体动态
- 地图/天气等工具页面
- 登录页/注册页

**建议**：
- 该页面不适合此类工具保存
- 如需保存信息，考虑截图或打印为 PDF

---

### PAGE_TOO_SHORT — 正文内容过短

**触发条件**：
- 提取出的正文不足 100 个字符

**典型场景**：
- 页面加载不全（网络慢、JS 未执行完）
- 页面仅包含一张图和一句话
- 页面被部分截断（如付费墙只显示了开头）

**建议**：
- 刷新页面后重试
- 检查是否需要滚动加载更多内容
- 检查是否有「展开全文」按钮未点击

---

### BLOCKED_BY_LOGIN — 需要登录/付费

**触发条件**：
- 检测到页面中存在 3 个以上的登录/付费关键词
- 同时页面正文内容很少（< 200 字符）

**典型场景**：
- 微信公众号文章（部分限制）
- 知乎盐选/付费专栏
- 各大新闻网站的付费墙（财新、FT中文网、36氪深度等）
- Medium 会员文章
- Substack 付费内容

**建议**：
- 先登录/订阅后再保存
- 如果已登录仍是此错误，请在对应网站手动复制内容

---

### EMPTY_PAGE — 页面内容为空

**触发条件**：
- 页面 body 内文本少于 50 字符

**典型场景**：
- 网站崩溃或返回空白页
- 网络请求失败（404、500 等）
- 需要 JavaScript 渲染的 SPA 页面在加载时就被尝试提取
- 被防火墙/WAF 拦截

**建议**：
- 刷新页面确认内容正常显示
- 检查网络连接
- 如果是 SPA 应用，等待几秒后再试

---

### CONTENT_SCRIPT_NOT_READY — 内容脚本未加载

**触发条件**：
- 扩展的内容脚本尚未注入到页面中
- 页面在 content script 加载完成前就收到了提取请求

**典型场景**：
- 刚打开的标签页（页面还在加载中）
- 浏览器扩展页面（`chrome://`、`about:`）
- 页面被浏览器安全策略阻止了脚本注入

**建议**：
- 刷新页面后重试
- 确保不是浏览器内部页面

---

### DOWNLOAD_FAILED — 文件下载失败

**触发条件**：
- `chrome.downloads.download()` API 调用失败

**典型场景**：
- 磁盘空间不足
- 指定保存路径不存在且无法自动创建
- 下载目录被系统或安全软件保护
- 文件名包含操作系统不允许的字符

**建议**：
- 检查磁盘空间
- 在设置页面修改保存路径（绝对路径，例如 `/home/bao/下载/文章`）
- 避免路径中包含特殊字符

---

### BLOCKED_BY_ROBOTS — 被网站反爬机制屏蔽

**触发条件**：
- 网站通过 robots.txt、JavaScript 挑战、CAPTCHA 等方式阻止访问

**典型场景**：
- 部分需要验证的网站
- CloudFlare 保护的页面
- 某些新闻网站对非浏览器请求的拦截

**建议**：
- 使用浏览器的「另存为」功能手动保存
- 或者在页面中手动复制内容

---

### UNSUPPORTED_URL — 不支持的链接类型

**触发条件**：
- URL 不是 `http://` 或 `https://` 开头

**典型场景**：
- `chrome://settings/`
- `file:///home/user/doc.html`
- `about:blank`
- `data:text/html,...`
- `view-source:https://...`

**建议**：
- 仅支持普通网页链接
- 浏览器内部页面不支持保存

---

### TIMEOUT — 操作超时

**触发条件**：
- 页面加载超过 18 秒
- 内容脚本未在预期时间内响应

**典型场景**：
- 网络缓慢
- 页面包含大量资源（图片、视频、广告追踪脚本）
- 网站服务器响应慢

**建议**：
- 检查网络连接
- 等待页面完全加载后再保存

---

### UNKNOWN — 未知错误

**触发条件**：
- 所有未归类到以上错误码的异常

**建议**：
- 尝试刷新后重试
- 查看 Chrome 扩展控制台日志（`chrome://extensions/` → 点击扩展的「背景页」）
- 如频繁出现，请报告给开发者

---

## 日志

所有保存操作（成功和失败）都会记录到扩展的本地存储中。

**查看日志**：点工具栏图标 → 设置 → 「保存日志」标签页

**导出日志**：在日志页点击「导出日志」，会生成一份 Markdown 格式的日志文件（包含链接、结果、错误码、时间），保存到下载目录。

**日志字段说明**：

| 字段 | 说明 |
|------|------|
| `time` | 操作时间（ISO 8601 格式） |
| `url` | 页面链接地址 |
| `title` | 文章标题 |
| `result` | 保存结果（成功/失败） |
| `errorCode` | 错误码（成功时为 `-`） |
| `errorDetail` | 错误详情 |

日志保留最近 200 条记录。
